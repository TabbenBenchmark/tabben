Collection of Datasets¶

class tabben.datasets.DatasetCollection(location, *names, split='train', download=True, transform=None, target_transform=None)¶

Represents a collection of tabular datasets, providing some convenience methods to bulk load, evaluate, or extract metadata/extras from a set of datasets.

classmethod match(location, *, task=None, outputs=None, classes=None)¶

Create a dataset collection where all the datasets match all given conditions. This can be used, for example, to get a collection of all binary classification datasets.

table(*columns)¶

Returns select attributes of the datasets in this collection in a pandas dataframe (note, this does not return the data attributes, but the meta-attributes of the datasets themselves, like task, number of examples, types of attributes, etc.).

Other Utilities¶

tabben.datasets.ensure_downloaded(data_dir: Union[str, bytes, os.PathLike], *datasets: str)¶

Downloads the specified datasets (all available datasets if none specified) into the data directory. This is useful e.g. using this package in an environment without Internet access or establishing local caches.

Parameters

  • data_dir – directory to save the dataset files into

  • *datasets – list of dataset names to load (if empty, as if all datasets)

tabben.datasets.register_dataset(name: str, task: str = 'classification', *, persist=False, **kwargs)¶

Add new datasets to the benchmark at runtime (after package loading).

Parameters

  • name – the name of the dataset (used as a primary index, cannot be ‘all’)

  • task – which task (see allowed_tasks)

Keyword Arguments

  • persist – whether to also save this dataset into the data file (only for this installation)

  • data_location (required) – URL pointing to the NPZ file for this dataset

  • outputs (recommended, defaults to 1) – number of output variables

  • classes (recommended for classification tasks, defaults to 2) – number of classes (if multiple output variables, must all have the same number)

  • **kwargs – all other keyword arguments will be added as additional metadata
